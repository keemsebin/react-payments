# react-payments

**기능**

- [x] 사용자가 입력하는 카드 번호를 실시간으로 파악 가능해야한다.
- [ ] Visa나 MasterCard에 해당하면 해당 브랜드의 로고를 UI에 표시한다.

**유효성 검사**

- [ ] 월과 년도를 범위 내에서만 입력할 수 있어야 한다.
- [x] 입력 제한을 두어 사용자가 숫자만 입력할 수 있도록 한다.

**컴포넌트**

- [x] Input
- [x] Text
- [x] Card
- [x] Layout
    - [x] AppLayout
    - [x] InputLayout
- [x] Flex
